# Rekan Setia

Fondasi pendamping harian yang ringan dan stabil. Dibangun dengan Next.js App Router, TypeScript, Tailwind CSS, dan next-themes.

## Instalasi

```bash
npm install
```

## Menjalankan development

```bash
npm run dev
```

Buka `http://localhost:3000`.

## Build dan start produksi

```bash
npm run build
npm run start
```

## Catatan PWA

Manifest dan ikon masih tersedia, tetapi service worker dinonaktifkan sementara untuk stabilisasi toolchain.
